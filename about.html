Game Description:

Player needs to match a pair of Marvel Avengers cards.

Turning over one card and then try to find a matching pair.

There are a total of 9 pairs, 18 cards total.


List of Skills To Build The MVP Game:

  • HTML
  • CSS Selectors, Cascading & Specificity
  • CSS Box Model, Margin & Dimensions
  • CSS Positioning & Floating
  • Javascript: Conditional Statements, Loops, DOM Creations
